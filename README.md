
# Weather Forecast Web Application

## Overview
A full stack web application that displays 5-day weather forecasts using real-time API data and map-based visualization.

## Features
- Location-based weather forecasting
- 5-day temperature and condition display
- Interactive map visualization
- Real-time API integration

## Tech Stack
- Backend: Python, Flask, Django
- Frontend: HTML, CSS, JavaScript
- APIs: Weather REST API
- Data Format: JSON

## How It Works
1. User selects or enters a location
2. Application calls a weather API
3. API response is processed on the backend
4. Weather data is displayed along with map visualization
